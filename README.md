# Spring Core Autowiring Demo

## Overview

This project demonstrates Dependency Injection (DI) in Spring Core using XML configuration. It covers various dependency injection techniques including Setter Injection, Constructor Injection, and Autowiring.

## Features

* Spring IoC Container
* XML-based Bean Configuration
* Setter Injection
* Constructor Injection
* Autowiring by Name
* Autowiring by Type
* Constructor Autowiring
* Primary Bean Configuration
* Autowire Candidate Configuration

## Technologies Used

* Java
* Spring Framework
* Maven
* Eclipse IDE

## Project Structure

```text
src/main/java
├── com.telusko.main
└── com.telusko.services

src/main/resources
└── applicationContext.xml
```

## Learning Outcomes

* Understanding Inversion of Control (IoC)
* Implementing Dependency Injection in Spring
* Configuring Beans using XML
* Working with Spring Autowiring mechanisms
* Managing bean selection using `primary` and `autowire-candidate`

## How to Run

1. Clone the repository.
2. Import the project into Eclipse or any Java IDE.
3. Update Maven dependencies.
4. Run the main class.
5. Observe dependency injection and autowiring behavior.
